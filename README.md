# Renewables Wizard

Open-source knowledge tool for solar, battery storage, EV charging, and microgrid planning.

**[Live Demo →](https://joshuastr.github.io/renewables-wizard/)**

## Features

- **Advisor** — Decision-tree wizard for system recommendations
- **Knowledge Base** — Searchable FAQ and technical guides
- **Calculator** — Solar sizing, EV infrastructure, and fleet ROI calculators
- **Product Comparison** — Side-by-side spec comparison with manufacturer and type filters
- **Incentive Finder** — US and Canadian rebate, tax credit, and grant database with filterable dropdowns

## Tech Stack

- Vanilla JavaScript (no framework)
- Vite for bundling and dev server
- Static JSON data (no backend required)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5174](http://localhost:5174)

## Data

All product specs, incentive programs, and decision trees are stored as JSON in the `data/` directory. Contributions welcome.
